# YICHEN LOGISTICS (YCL) - Ocean Freight Comparison System

## Project Overview
This project is a React-based web application for international ocean freight rate comparison and booking for YICHEN LOGISTICS. It simulates a platform where users can search for freight rates between ports, view details, and pay points to unlock contact information for booking.

## Key Features
- **User Authentication**: Login/Register with simulated SMS/Email verification; New users get 10 points free.
- **Referral System**: Invite friends to register, both parties get points rewards; Relationship mapping.
- **User Center**: Manage account, view invitees, check points history.
- **Search Engine**: Search freight rates by origin and destination ports.
- **Points System**: Consume 3 points to view contacts; Consume 2 points to publish; Recharge simulation.
- **Penalty System**: Terms of Service includes 15 points deduction rule for fake info.
- **Inventory Management**: Sold out slots prevent booking.
- **Responsive Design**: Mobile-first, luxury/minimalist UI using TailwindCSS.
